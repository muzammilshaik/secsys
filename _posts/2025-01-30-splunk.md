Splunk Queries for SOC Analyst
===========================================================================

# üîç Advanced Splunk Queries for Security Monitoring

As a System Engineer, monitoring security events is critical for ensuring system integrity and identifying potential threats. Here‚Äôs a collection of essential Splunk queries categorized for different scenarios, from failed login attempts to suspicious network traffic.

## üîê Authentication Attempts

### üîê Failed Login Attempts  
```
sourcetype=auth* "authentication failure"
| stats count by user
| sort -count
```

### üîë Failed SSH Attempts  
```
sourcetype=linux_secure "Failed password for"
| stats count by src_ip
| sort -count
```

### ‚úÖ Successful SSH Attempts  
```
sourcetype=linux_secure "Accepted publickey for"
| stats count by src_ip
| sort -count
```

### üåé Successful Login Attempts from New or Unknown IP Addresses  
```
sourcetype=access_* action=login
| stats count by user, src_ip
| where count=1
```

### üîì Account Takeover Attempts
``` 
sourcetype=access_* action=login
| stats count by user
| where count > 10
```

### üõ†Ô∏è Brute Force Attacks on SSH Servers  
```
sourcetype=linux_secure action=invalid
| stats count by src_ip
| where count >= 10
```

###  üìß Brute Force Attacks on Email Accounts  
```

| stats count by src_ip
| where count >= 10
```

### üîê Brute Force Attacks on SSH Servers  
```

| stats count by src_ip
| where count >= 10
```

###  üîê Brute Force Attacks on SSH Servers (Failed Login Attempts)  
```

| stats count by src_ip
| where count >= 10
```

###  üõ†Ô∏è Brute Force Attacks on MSSQL Servers  
```

| stats count by src_ip
| where count >= 10
```

###  üõ†Ô∏è Brute Force Attacks on RDP  
```

| search Logon_Type=10 AND Status="0xC000006D"
```

### üñ•Ô∏è Privilege Escalation Attempts on Windows Systems  
```

| search Image_Path="*\\System32\\*" AND NOT User="SYSTEM"
```

###  ‚û°Ô∏è Lateral Movement Attempts Using Remote Registry  
```

| search Object_Name="*\\REGISTRY\\MACHINE\\SOFTWARE" AND NOT User="SYSTEM" AND
NOT User="NETWORK SERVICE" AND NOT User="LOCAL SERVICE"
```


###  ‚û°Ô∏è Lateral Movement Attempts Using SMB  
```

| search Object_Name="*\\ADMIN$" OR Object_Name="*\\C$"
```

```

```

###  ‚û°Ô∏è Lateral Movement Attempts Using RDP (Successful Logins)

```

### üñ•Ô∏è Privilege Escalation Attempts on Windows Systems  
```

| search "Task Scheduler service found a misconfiguration" AND NOT User="SYSTEM"
```

### üñ•Ô∏è Privilege Escalation Attempts on Windows Systems  
```

| eval user_account=mvindex(Account_Name,1)
| search "Security ID" NOT IN ("SYSTEM","LOCAL SERVICE","NETWORK SERVICE")
```

## üõ°Ô∏è Security Threats
#### üõ°Ô∏è Security Threats  
```

| eval password_length=length(password)

```

###  üö® Traffic to Known Malicious IP Addresses

###  ü¶† Malware Infections

###  üëÄ Insider Threats

```

| search extension="encrypted" OR extension="locked" OR extension="ransom"

```

###  ‚öîÔ∏è DDoS Attacks
```

| stats sum(bytes) as total_bytes by src_ip
| where total_bytes > 100000000
```

###  üñ•Ô∏è Privilege Escalation Attempts on Windows Systems

```

###  üåè Successful SSH Logins from Unusual Countries
```

| iplocation src_ip
| stats count by src_country
| where count > 10 AND NOT src_country="United States"
```

###  üñ•Ô∏è Ransomware Activity on Windows Systems

rex field=Object_Name "\\\\.*\\\\(?<filename>.+)"
| rex field=filename ".*\\.(?<extension>[^\\.]+)"
| search extension="encrypted" OR extension="locked" OR extension="ransom"
